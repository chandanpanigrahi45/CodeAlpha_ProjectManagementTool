# TaskFlow — Project Management Tool

A collaborative project management web application built as part of the CodeAlpha Full Stack Development Internship.

## Features
- Kanban board with 4 columns (To Do, In Progress, Review, Done)
- Create, view, and delete tasks
- Add comments to tasks
- Set priority levels (High, Medium, Low)
- Assign tasks to team members
- Set due dates and task tags
- Live stats dashboard (total, done, in progress, high priority)
- Project and team member sidebar navigation

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (DOM Manipulation)
